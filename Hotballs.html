Hot Balls

(May 12, 2026)

Click image to play.

Hot balls is a mini video game I created using html coding. It follows the format of a cookie clicker game but this one is basketball themed. The goal is to click or “shoot” baskets and everytime you score, you gain a point. The percentage to actually make a basket is quite low so it’ll take time. With the game, you get an item shop where you get upgrades and fun little collectables. You get items like throwable objects which then show up being thrown when you buy them (Purple), better accuracy items (Teal), more points items (Red), and collectables (Golden). I used hyperlinks and html coding as techniques to make this game. I illustrated all the photos in the game using procreate on my Ipad and imported them into my computer. The sprites were originally little animations I made for the game and then I converted them to left and right 3-framed sprites. I was curious on how I would make a videogame through HTML and what would all the properties be formatted like? I looked up alot of codes on google specifically for save properties and loading as well. Making this project is important to me because I’m starting to learn more about coding even though I’m still a beginner. It’s fun for me to make something and see the outcome on a screen and I was really passionate about making this mini game for the sake of knowing a little bit about how to code. Also I’m impressed with myself for drawing everything in a timely manner, especially the main menu. I made the items really expensive to purchase so it’ll take awhile before maxing out the item shop.

I was thinking about Mark Tribe for this project because he’s explained how new media would take over the traditional ways of art. I don’t really like it but it seems to be true. It inspired me since the whole point of my project is to use digital arts for it. Therefore, it was all done with New media. Another Artist that inspired me was Laurel Schwulst. In their essay “My website is a shifting house”, they explained the possibilities of what your website can look like. I like how creative it was and reading through it. Although this project I made isn’t exactly a website, I still used html to make it which is what you’d need to make a website.
